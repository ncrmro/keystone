# Desktop

Keystone Desktop is the graphical layer of Keystone OS.

It combines [Hyprland](https://wiki.hypr.land/Getting-Started/Installation/), [Walker](https://github.com/abenz1267/walker), [Waybar](https://github.com/Alexays/Waybar), [Ghostty](https://ghostty.org/docs), notifications, screenshots, and
project-aware desktop navigation into a keyboard-first environment.

## Getting started

Start here if you are new to Keystone Desktop:

1. Read [Navigation](navigation.md)
2. Read [Desktop Keybindings](desktop/keybindings.md)
3. Read [Walker](desktop/walker.md)
4. Configure your displays with [Monitors](desktop/monitors.md)
5. Set up biometric unlock with [Fingerprint Unlock](desktop/fingerprint-unlock.md) if your hardware supports it

## What Keystone Desktop includes

- Hyprland for the compositor and window management
- Ghostty as the default terminal
- Walker and Elephant for launcher and project menus
- Waybar for status and workspace display
- Mako for notifications
- clipse and `wl-clipboard` for clipboard history
- Screenshot and screen recording tooling

## Daily workflow

The intended workflow is:

- use Ghostty for shell access,
- use Zellij inside Ghostty for long-running project sessions,
- use Walker for app and project switching,
- use `pz` for note-backed project session management, and
- use Waybar and notifications for lightweight system feedback.
